Web Service: общие объекты, которые могут передаваться в качестве аргументов и возвращаться из функций ( интерфейсной части ).
Обязательно: при использовании на клиенте данные объекты должны должны быть классическими POJO (constructor without arguments, get-set methods for all fields) полное имя объекта ( пакет+имя класса) должно совпадать на клиенте с серверной частью
